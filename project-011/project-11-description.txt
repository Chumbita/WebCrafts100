Proyecto 11:
Este proyecto consiste en una página web diseñada para el inicio de sesión de usuarios, desarrollada exclusivamente con HTML y CSS. Esta página de inicio de sesión se ha creado como parte de un proyecto inicial sin funcionalidad ni almacenamiento de datos. Su propósito principal es servir como una base sólida para proyectos futuros que requieran características adicionales, como la funcionalidad de registro de usuarios y la implementación de almacenamiento de datos, así como también la posibilidad de iniciar sesión o  registrarse con google. La página de inicio de sesión presenta un diseño limpio y minimalista, con campos para ingresar el nombre de usuario o correo electrónico, la contraseña y un botón para iniciar sesión. Además, se incluye la opción de iniciar sesión con Google para mejorar la accesibilidad y la comodidad del usuario.

Project 11:
This project consists of a web page designed for user login, developed exclusively with HTML and CSS. This login page has been created as part of an initial project without functionality or data storage. Its main purpose is to serve as a solid foundation for future projects that require additional features, such as user registration functionality and data storage implementation, as well as the ability to log in or register with Google. The login page features a clean and minimalist design, with fields to enter the username or email, password, and a login button. Additionally, the option to log in with Google is included to enhance accessibility and user convenience.
