Proyecto 6:
Nuestro proyecto consiste en crear una sección de pestañas que incluye numerosas pestañas, cada una con un título y el texto correspondiente. Al hacer clic en una pestaña, el título se cambiará en consecuencia. También agregaremos un elegante efecto de desplazamiento a cada pestaña, y cuando un usuario seleccione una pestaña, su color de fondo cambiará. Para implementar esto, usaremos JavaScript para actualizar dinámicamente el contenido. Diseñaremos las pestañas y crearemos el efecto de desplazamiento usando CSS y la pseudoclase :hover para aplicar diferentes estilos cuando el usuario pase el cursor sobre cada pestaña.

Project 6: 
Our project involves creating a tab section that includes numerous tabs, each with a title and corresponding text. When you click on a tab, the title will be changed accordingly. We will also add an elegant hovering effect to each tab, and when a user selects a tab, its background color will change. To implement this, we will use JavaScript to dynamically update the content. We will style the tabs and create the hovering effect using CSS and the :hover pseudo-class to apply different styles when the user hovers over each tab.
