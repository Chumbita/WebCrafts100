Proyecto 10: 
Este proyecto presenta una barra lateral dinámica donde los usuarios pueden navegar haciendo clic en diferentes botones. Cada botón activa un efecto "Activo" para indicar la sección actual que está viendo el usuario. Además, hay un menú estilo hamburguesa con animaciones de apertura y cierre. Al hacer clic en el botón de menú hamburguesa, la barra lateral se muestra o se oculta. Esta barra lateral contiene íconos junto con texto que describe el contenido de cada sección. La idea es que al hacer clic en el botón del menú, la barra lateral aparezca si está oculta y desaparezca si está visible, permitiendo al usuario navegar a través de las diferentes secciones de la aplicación según su preferencia.

Proyect 10: 
This project features a dynamic sidebar where users can navigate by clicking on different buttons. Each button activates an "Active" effect to indicate the current section the user is viewing. Additionally, there is a hamburger-style menu with opening and closing animations. Clicking on the hamburger menu button toggles the sidebar, showing or hiding it. The sidebar contains icons accompanied by text describing the content of each section. The idea is that clicking on the menu button will reveal the sidebar if it's hidden and hide it if it's visible, allowing users to navigate through the different sections of the application according to their preference.
