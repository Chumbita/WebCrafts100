Proyecto 5: 
Este proyecto tiene como objetivo crear un convertidor de temperatura. La versión final del proyecto incluye tres entradas para los formatos de temperatura Celsius, Fahrenheit y Kelvin. Cuando cambie cualquiera de estas entradas, por ejemplo, si elige 100 grados Celsius, verá el valor equivalente de Celsius en formatos Fahrenheit y Kelvin. Además, si cambia una entrada, verá un cambio en tiempo real dentro de las otras entradas. Por ejemplo, si cambias la entrada de Fahrenheit a 100 grados, verás que es igual a 37 grados Celsius y 311 Kelvin. Para crear este proyecto, aprenderemos a usar un detector de eventos "onchange" para realizar un seguimiento de los cambios dentro de las entradas. También usaremos la instrucción switch para realizar un seguimiento de los cambios en función del nombre de la entrada en la que estamos trabajando. Además, usaremos CSS para darle al proyecto un aspecto moderno. En resumen, este proyecto nos ayudará a aprender a crear un convertidor de temperatura usando JavaScript y CSS, y aprenderemos a rastrear los cambios usando detectores de eventos y sentencias switch.

Project 5:
This project aims to create a temperature converter. The final version of the project includes three inputs for Celsius, Fahrenheit, and Kelvin temperature formats. When you change any of these inputs, for example, if you choose 100 degrees Celsius, you will see the equivalent Celsius value in Fahrenheit and Kelvin formats. Additionally, if you change one input, you will see a real-time change inside the other inputs. For instance, changing the Fahrenheit input to 100 degrees will show you that it is equal to 37 degrees Celsius and 311 Kelvin. To create this project, we will learn how to use an "onchange" event listener to track changes inside the inputs. We will also use the switch statement to track changes based on the name of the input we are working on. Additionally, we will use CSS to give the project a modern look. In summary, this project will help us learn how to create a temperature converter using JavaScript and CSS, and we will learn how to track changes using event listeners and switch statements.
