Proyecto 9
Cree una calculadora elegante y moderna con todos los botones necesarios, incluidos números, operadores, signo igual. El usuario escribe cualquier expresión aritmética y la calculadora la resuelve al instante. Usé la función 'eval' de JavaScript para evaluar las expresiones. Los botones tienen efectos de desplazamiento y sombras dentro para mejorar el diseño. Este proyecto fue el más desafiante de todos los que hice hasta ahora, me llevó aproximadamente 7 horas el pensamiento de la lógica y la programación en JS. La calculadora no fue testeada por completo así que es posible que todavía existan algunos errores y bugs por lo que es posible que este proyecto se esté actualizando ante un feedback de error o bug.

Project 9
I create a sleek and modern calculator with all the necessary buttons, including numbers, operators, equal sign. The user types in any arithmetic expression and the calculator solves it instantly. I used JavaScript's 'eval' function to evaluate expressions. The buttons have scrolling effects and shadows inside to enhance the design. This project was the most challenging of all the ones I did so far, it took me about 7 hours thinking about logic and programming in JS. The calculator was not fully tested so it is possible that there are still some errors and bugs so it is possible that this project is being updated due to a bug feedback.
